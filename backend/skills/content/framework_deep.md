## 深调研独有：4 个增强维度

### 增强 1：多视角交叉验证

**目标**：同一个事实，从**至少 2 个不同视角**评估。

**操作**：
- 视角 A：用户视角（基于 `relationship`）
- 视角 B：行业视角（基于 `audience`）
- 视角 C：竞争视角（基于 `focus_dimensions`）
- **如果 3 个视角结论一致 → 置信度高**
- **如果 3 个视角结论冲突 → 标记为"需进一步研究"**

**输出模板**：
```markdown
### 多视角交叉验证

| 视角 | 结论 | 置信度 |
|---|---|---|
| 用户视角 | {结论} | {高/中/低} |
| 行业视角 | {结论} | {高/中/低} |
| 竞争视角 | {结论} | {高/中/低} |
| **综合判断** | {最终结论} | {高/中/低} |
```

### 增强 2：历史脉络与对比

**目标**：把当前事件放在**历史脉络**里看。

**操作**：
- 找 1-2 个**历史可比事件**（同公司 / 同赛道 / 同技术）
- 对比当时的解读 vs 实际演化
- 评估今天事件的**真实重要性**（避免过度解读或忽视）

**输出模板**：
```markdown
### 历史脉络

- **历史可比事件 1**：[描述]
  - 当时解读：[大家怎么看]
  - 实际演化：[结果如何]
  - 对今天的启示：[类比意义]

- **历史可比事件 2**：...
```

### 增强 3：跨领域影响分析

**目标**：这件事**不只影响调研对象本身**，还影响哪些相关方？

**操作**：
- 列出至少 3 个**间接受影响**的群体
- 每个群体评估影响类型和强度

**输出模板**：
```markdown
### 跨领域影响

| 受影响方 | 影响类型 | 影响强度 |
|---|---|---|
| [群体 1] | [正面/负面/中性] | [高/中/低] |
| [群体 2] | ... | ... |
| [群体 3] | ... | ... |
```

### 增强 4：反面观点与反驳

**目标**：列出**最有力的反面观点**，并诚实评估。

**操作**：
- 至少 1 条"聪明人会怎么反驳这个分析"
- 评估反驳是否成立
- 给出**自己的最终判断**

**输出模板**：
```markdown
### 反面观点

- **反驳 1**：[聪明人会怎么质疑这个分析]
  - 是否成立：[部分/完全/不成立]
  - 我的回应：[怎么解释/承认/调整]

### 最终判断

{基于多视角 + 历史 + 跨领域 + 反面观点，给出最完整的判断}
```

## 深调研的适用 vs 不适用

### 适合深调研

- ✅ 写一篇行业分析文章
- ✅ 准备投融资材料
- ✅ 给老板/客户的 brief
- ✅ 重大决策前的尽调
- ✅ 写 PRD 前的市场调研

### 不适合深调研（用浅调研）

- ❌ 临时想了解一个概念（5 分钟）
- ❌ 已经知道答案只是想确认
- ❌ 闲聊式问题

## 质量自检（深调研加强版）

- [ ] 多视角交叉验证是否覆盖 3 个视角？

- [ ] 是否识别 3+ 个跨领域受影响方？
- [ ] 是否至少给出 1 个聪明人的反驳？
- [ ] 报告字数是否在 1500-2500 字之间？
- [ ] 是否避免"标题党"和"营销话术"？
